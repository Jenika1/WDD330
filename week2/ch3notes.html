Arrays, Logic, and Loops - Ch. 3

Arrays 

  • Ordered list of values.
  • const myArray = []; or const myArray = new Array();
  • const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk'];
  • You don’t even have to use the same types of items inside an array

Example: Delete avengers[3]; 

  • The fourth entry, 'Hulk' (with an index of 3), has been removed, but it has been replaced by a value of undefined:
    avengers;
    << ['Captain America', 'Iron Man', 'Thor', undefined]

Destructuring - taking values out of an array and presenting them as individual values.

  • Allows you to assign multiple values at the same time, using arrays
  • Example: const [x,y] = [1,2]; This will return x << 1 and y << 2

Array Properties and Methods

Length property:

  • avengers.length;
    << 6
  • To find the last item in an array use subtract 1
    • Example:  avengers[avengers.length - 1];
      << 'Hulk'
  • The length property is mutable (you can manually change it)

Pop, Push, Shift, and Unshift

pop() method:

  • Removes the last item from an array completely.

shift() method:

  • removes the first item in the array

push() method:

  • appends a new value to the end of the array.

unshift() method:

  • this appends a new item to the beginning of the array.

Merging Arrays

concat() method

  • creates another array and combines the two arrays.

spread operator (...)

  • placed in front of an array
  • spreads out the elements of that array

The join() Method

  • turns the array into a string made up of all the items in the array, separated by commas.
  • You can choose a separator other than a comma by placing it inside the parentheses. Eample: avengers.join(' & ');
    << 'Captain America & Iron Man & Thor & Hulk & Hawkeye & Black Widow'

Slicing and Splicing

slice() method:

  • subarray
  • chopping out a slice of anoriginal array, starting at one position and finishing at another.
  • non-destructive â€• no items are actually removed from the array.

splice() method:

  • removes items from an array then inserts new items in their place.
  • destructive - changes the value of the array permanently.
  • Example: avengers.splice(3, 1, 'Scarlet Witch');
    • (the index at which to start the splice, how many items to remove, Every value after this is then inserted into the array in the same place the other items were removed.)

reverse() method:

  • Reverses the order of the array permanently.

sort() method:

  • sorts the array in alphabetical order.

indexOf() method:

  • find the first occurrence of a value in an array
  • it will return the index of the first occurrence of that item
  • If the item is not in the array, it will return -1

includes() method:

  • returns a boolean value depending on whether the array contains a particular element or not.
  • You can also add an extra parameter to indicate which index to start the search from. Example: avengers.includes('Black Widow', 1); // will start the search from the second element in the array.

Multidimensional Arrays

  • array of array's
  • use two indices: one to refer to the item’s place in the outer array, and one to refer to its place in the inner array
    • Example: const coordinates = [[1,3],[4,2]];
      coordinates[1][0]; // The first value of the second array
      << 4

Sets

  • Introduced to the specification in ES6
  • A set is a data structure that represents a collection of unique values.
  • Cannot include any duplicate values.

Creating Sets

  • const list = new Set();
  • no literal notation for creating sets
