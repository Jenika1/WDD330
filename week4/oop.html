Object-Oriented Programming

  • a style of programming that involves separating the code into objects that have properties and methods.
  • There are three main concepts in OOP: encapsulation, polymorphism, and inheritance.

Encapsulation

  • the inner workings are kept hidden inside the object and only the essential functionalities are exposed to the end-user, such as the “on” button.

Polymorphism

  • the same process can be used for different objects.

Inheritance

  • taking the features of one object then adding some new features.

Constructor Functions

  • An alternative way to create objects is to use a constructor function
